# @eddyq/queue

Node.js client for [eddyq](https://github.com/eddyqueue/eddyq) — a Rust + Postgres job queue.

**Pre-alpha.** Real bindings land in Phase 3 of the v1.0 roadmap.

## Install

```bash
pnpm add @eddyq/queue
```

We strongly recommend **pnpm** or **yarn** over npm — see the [npm caveat in the docs](https://github.com/eddyqueue/eddyq) for why.
